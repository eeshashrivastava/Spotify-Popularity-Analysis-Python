# Spotify Popularity Analysis

This project explores the relationship between Spotify audio features and track popularity using Python.

## Project Overview
- Analyzed over 110,000 Spotify tracks
- Examined how features such as energy, danceability, tempo, and valence relate to popularity
- Compared audio characteristics across popularity buckets and genres

## Project Structure
- `data_raw/` contains the original dataset
- `data_processed/` contains the cleaned dataset used for analysis
- `notebooks/` contains data loading, EDA, and modeling notebooks

## Tools Used
- Python
- pandas
- numpy
- matplotlib
- seaborn
